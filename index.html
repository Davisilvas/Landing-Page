Stacks

HTML

É a linguagem de marcação usada para dar estrutura ao site.

CSS

É a linguagem de estilização responsável por dar estilo ao HTML.

Javascript

Linguagem de programação usada tanto no front quanto no back, famosa por proporcionar interação entre o sistema e o usuário.

Node.Js

Ambiente para execução de código javascript que é usado para rodar no server side, sendo assim o Js no backend.

React Js

Biblioteca javascript para criação de interfaces de usuário.

PHP

Linguagem de programação server side, capaz de fazer as interações entre cliente, servidor e banco de dados.

MySql

Sistema de gerenciamento de banco de dados relacional..

Git

Sistema de controle de versão de códigos.

Python

Linguagem de programação orientada a objetos que ajuda em data science, machine learning, automação de scripts e mais.
